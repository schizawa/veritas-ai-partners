Founder

代表プロフィール

遅沢翔
Sho Chizawa
Founder & CEO — Veritas AI Partners / フェリデンシアキャピタル代表取締役

20代で起業。EC事業・産業用3Dプリンター事業・デンタルテクノロジー事業など複数事業を立ち上げ、恵比寿ガーデンプレイスタワーに本社を構える。経営者でありながら、自らコードを書き、AIシステムを設計・実装・運用するという極めて稀なバックグラウンドを持つ。

「経営者がAIを理解していないと、正しい投資判断ができない」という信念のもと、Claude Opus 4 / Sonnet 4（Anthropic）を日常の開発環境として使いこなし、GPT-4.1 / o3-pro（OpenAI）の高度推論、Gemini 2.5 Pro（Google）の100万トークンマルチモーダル分析、Grok 3（xAI）のリアルタイム推論、DeepSeek-V3 / R1のオープンソースモデルまで、主要LLMの全てを実務で検証し使い分ける。

コード生成だけでなく、ComfyUIによるFlux.1 / Stable Diffusion 3のローカル画像生成パイプライン、LoRA / DreamBoothによるカスタムモデル学習、Kling 2.0 / Runway Gen-3 / Sora（OpenAI）による動画生成、ElevenLabs Turbo v3 / OpenAI Whisper / Sesame CSMによる音声合成・理解まで、生成AIのフルスタックを実務で運用。

この「経営 × AI実装」の二刀流が、Veritas AI Partnersの競争優位の源泉。提案書を書くコンサルタントではなく、自分で動くシステムを作れる経営者が、あなたの事業変革をリードする。

Our AI Philosophy

AIへの取り組み方

🧪 LLMは「使い分け」が全て

Claude Opus 4 — 高度なコード生成とエージェント設計。Sonnet 4 — 高速かつ高精度な日常タスク。GPT-4.1 — 指示追従とコーディング。o3-pro — 数学的推論と複雑な論理。Gemini 2.5 Pro — 100万トークンの長文脈マルチモーダル分析。Grok 3 — リアルタイム情報とDeep Search。DeepSeek-V3 — コスト効率の良いオープンソース。適材適所のモデル選定が、品質とコストの最適解を生む。

🤖 エージェント設計の時代

単発のプロンプトではなく、Tool Use / Function Calling / MCP（Model Context Protocol）を活用したAIエージェントを設計。複数のAPIを自律的に呼び出し、判断し、実行するシステムが、人間の介入なしに業務を完遂する。Claude Code Agent SDKによるマルチエージェント並列処理も実戦投入済み。

📊 RAGで企業固有の知識を武装

Retrieval-Augmented Generation（RAG）により、お客様の社内文書・FAQ・過去の対応履歴をベクトルデータベース（Supabase pgvector / Pinecone）に格納。汎用LLMでは答えられない、御社固有の質問に正確に回答するAIを構築。ハルシネーション率を大幅に低減。

🎨 生成AIでクリエイティブを量産

ComfyUIによるノードベースの画像生成ワークフローで、商品画像の背景差し替え・バリエーション生成を自動化。LoRA（Low-Rank Adaptation）で商品のスタイルを学習させ、ブランド一貫性のある画像を量産。動画はKling 2.0のモーション制御で商品紹介映像を生成。
