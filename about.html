Why we exist

>> Three convictions that shaped this firm.

// principle 01

提案書はゴミになる

100ページのAI戦略資料は、半年後には誰も開きません。残るのは動くシステムだけです。だから私たちは、最初からコードを書きます。

// principle 02

経営者がコードを書く

代表自身が現役の実装者です。Claude Code を中心に Cursor / Codex CLI / 自作 MCP サーバー群を使い、社内では 20 以上の AI システムが月間数千万トークン規模で毎日動いています。「何が欲しいのか」を経営者の言葉で理解して、そのままコードにできるのが強みです。

// principle 03

中小企業に最適化する

エンタープライズ向けの大掛かりなAI基盤は必要ありません。1人〜数十人の事業にちゃんと効く、小さくて強いAIを最短で本番に載せます。
