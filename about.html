Why we exist

>> Three convictions that shaped this firm.

// principle 01

提案書はゴミになる

100ページのAI戦略資料は、半年後に誰も開かない。動くシステムだけが残る。だから我々は最初からコードを書く。

// principle 02

経営者がコードを書く

代表自身が現役プレイヤー。Claude Code を中心に、Cursor / Codex CLI / 自作 MCP サーバー群を束ね、社内では 20 以上の AI システム・月間数千万トークン規模の本番運用が日々回り続けている。クライアントの「何が欲しいのか」を、エンジニアではなく経営者の言語で理解し、即コードに落とす。

// principle 03

中小企業に最適化する

エンタープライズ向けの大袈裟なAI基盤は要らない。1人〜数十人の事業に効く、最短で本番投入できる小さく強いAIを作る。
